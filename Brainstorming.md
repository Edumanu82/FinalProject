# TOP 3 IDEAS WITH EVALUATIONS

## 1. Astronomy App

**Mobile:**
- Uses camera/library to post photos for others to see
- Uses location and sensors for real-time sky tracking  
- Strong mobile experience (akin to Instagram)

**Story:**
- Helps users explore and understand space  
- Fun, educational, and visually engaging
- Easy to use 

**Market:**
- Niche but global audience  
- Appeals to students and hobbyists  

**Habit:**
- Medium usage (mostly at night or during events)  
- Users return for stargazing sessions or interest in posting

**Scope:**
- Medium to high difficulty  
- Basic version of login page, post pages, user profiles, upcoming events, being able to post(camera/library), and pull from nasa api to show phtots

## 2. Car and Coffee Map

**Mobile:**
- Uses maps and location services  
- Real time updates for events  
- Designed specifically for mobile use  

**Story:**
- Connects car enthusiasts to local meets  
- Unique and community driven idea  

**Market:**
- Smaller niche audience  
- Strong engagement within car community  

**Habit:**
- Medium usage (event based)  
- Users check regularly for new meets  

**Scope:**
- Medium difficulty  
- Needs map and event posting features  

## 3. Netflix Clone

**Mobile:**
- Mainly video streaming (could be a website too)  
- Push notifications for new content  

**Story:**
- Easy to understand but not unique  
- Lacks originality compared to others  

**Market:**
- Very large market  
- Extremely competitive  

**Habit:**
- High usage potential  
- Users consume content frequently  

**Scope:**
- Very high difficulty  
- Hard to build fully within limited time  
- Basic version may still be complex
